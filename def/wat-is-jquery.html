Wat is jQuery?

In het kort

Programmeren vereist veel code invoeren, en dat kost veel tijd. Met JQuery wordt het ontwikkelproces gestroomlijnd, waarbij je controle hebt over zaken als:

  • Manipuleren van HTML/DOM
  • Manipulatie van CSS
  • HTML evenement methoden
  • Effecten en animaties
  • AJAX
  • Utilities

Portabiliteit maakt een belangrijk deel uit van JQuery. Dit betekent dat het werkt in alle bekende browsers met een uniforme functionaliteit. De library valt onder de OpenJS Foundation, waar ook Appium, Node.js toe behoren.

Ontwikkelen met jQuery

Er is zowel een gecomprimeerde als een ongecomprimeerde versie beschikbaar. De ongecomprimeerde versie is met name geschikt tijdens ontwikkeling of het opsporen van fouten. Het gecomprimeerde bestand bespaart bandbreedte en verhoogt de prestaties. Medio mei 2019 werd JQuery naar schatting gebruikt op 73% van alle websites op het internet. Daarmee is het veruit de meest gebruikte JavaScript library.


Plugins

Er zijn duizenden plugins verkrijgbaar om de functionaliteit verder uit te breiden, en specifieke toepassingen sneller te realiseren. Denk aan Ajax helpers, Web diensten, datagrids, dynamische lijsten, drag-and-drop of het gebruik van cookies.
